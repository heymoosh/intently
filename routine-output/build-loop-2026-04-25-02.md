## Key findings (full detail in RN-PORTING-OBSTACLES.md)

- **Most RN constructs are pure THROWAWAY.** `<View>`, `<Text>`, `<Pressable>`, `<ScrollView>` (vertical), `<TextInput>`, `<Modal>`, `<ActivityIndicator>`, `StyleSheet.create`, `react-native-svg`, `Platform.OS === 'web'` branches, `expo-google-fonts` + `useFonts`, `expo-status-bar` — all already covered by plain-React/DOM equivalents already running in `web/`.
- **`tokens.ts` is confirmed THROWAWAY (resolves PORTING.md OQ1).** `web/intently-tokens.jsx` ships every field in browser-native shape: shadows are CSS strings, fonts are font-family strings, spacing is `T.space(n)`. Only mechanical change is the namespace rewrite (`t.colors.X` → `T.color.X` — note capitalization + the singular `color` namespace).
- **`PainterlyBlock` + `MorningLight` are 1:1 duplicates.** `app/components/painterly/*.tsx` and `web/intently-imagery.jsx` use the same blob recipe + horizon SVG. Pure THROWAWAY.
- **`GoalDetail` / `ProjectCard` / `ProjectDetail` are mostly THROWAWAY.** Prototype already has them (`intently-flows.jsx:120-262`, `intently-projects.jsx:92-172`). Only the goal-back-chip in ProjectDetail needs porting (~10 LOC).
- **`BriefFlow` / `ReviewFlow` are MIXED.** Prototype's flows (`intently-flows.jsx:454-1027`) cover the full visual + script-machine. SATURDAY-PORT-TASK is wiring `onAccept` to `runAgent` + injecting Sam-flavored scripts + adding `agentRunning` / `agentError` props.
- **`VoiceModal` is the highest-leverage SATURDAY-PORT-TASK.** Prototype's `HeroListening` (`intently-hero.jsx:226-313`) renders the listening UI but uses a fake setInterval transcript. Wiring `useVoiceInput` (Web Speech API) + `classifyTranscript` POST is the ~80 LOC port that turns it real.
- **`App.tsx` wiring layer is the densest port-task.** `runAgent` + 3 `handleGenerateLive*` + `derivePhase` + URL phase override + 21-slot rotation pager + keyboard arrow-key nav + weekly-auto-fire latch. None of these exist in prototype today; ~200 LOC of pure-JS wiring to lift over.

4. **`expo-google-fonts` → Google Fonts `<link>` tags.** Verify `web/index.html` already has them; if not, 1-line addition.
5. **`react-native-web` `backgroundImage` casts** are non-issues on plain JSX. Mentioned for completeness.
