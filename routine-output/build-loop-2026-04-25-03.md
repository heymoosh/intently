- **Manual entity CRUD has no `app/lib/` helper.** Six AddZone destinations (goals, projects, journal, plan items, admin reminders, project todos) all flow through in-memory `useManualAdds` today. `app/lib/` ships zero entity-insert helpers. Documented as OPEN Q2.

- **OAuth / connections page is mocked end-to-end.** `OAuthFlow` simulates consent → auth → success with `setTimeout`. PORTING.md surfaces no OAuth lib. Defer for the demo.
