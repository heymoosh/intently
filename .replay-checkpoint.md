- **Responsive shell**: `Phone` wrapper drops the iOS bezel below 640px; `top: 62px` content offset becomes `env(safe-area-inset-top)`. `@media (max-width: 640px)` rules drop intro-note + dev-panel and let app fill viewport.
- **Persistence**: `useManualAdds` round-trips to `localStorage` key `intently:manual-adds:v1`. Day-state persisted at `intently:day-state:v1` keyed by ISO date. `?reset=1` clears both.
- **Day-state inference**: `inferInitialDayState()` reads clock — pre-7pm → `morning-empty`/`planned`; 7pm+ → `evening`. Saved value for today wins.
- **Dev panel gating**: hidden by default; shown only with `?dev=1` (or `intently:dev` localStorage flag set).
- **Typed Entry schema** (`makeEntry` in `intently-manual-add.jsx`): `{id, at, text, kind, source, glyph, ...extra}`. `pickGlyphFor(text, kind)` keyword-picks from the canonical glyph set.
- **`classify(text)` + `applyClassification(intent, m)`** in new file `intently-agent.jsx`: routes by keyword (morning:/afternoon:/evening: → plan, "remind me to" → reminder, "goal:" → goal, default → journal). Strips prefixes, returns clean `{kind, band?, body, reason}`.
- **`AgentComposer` overlay** in `index.html`: opens on hero radial pick `text`/`chat`. Live ConfirmationCard preview (tinted by kind, picked glyph, routing reason). Confirm → `applyClassification` → store mutator → localStorage.
- **Brief/Review write typed Entries**: `onBriefComplete(plan)` writes `kind:'brief'` Entry + seeds `MOCK_PLAN.bands` items into store as `addPlanItem(..., {source:'agent'})`. `onReviewComplete()` writes `kind:'review'` Entry.
- **Past → Day rail** renders typed metadata: glyph + kind eyebrow + time + source, tinted by kind (lilac=journal, sage=brief, dusk=review).
