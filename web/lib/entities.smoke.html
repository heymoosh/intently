entities.js — manual smoke aid

Open the devtools console. Each helper is on window. Try things like:

await window.insertGoal('test goal from smoke')
await window.listGoals()

await window.insertProject('test project')
const projects = await window.listProjects()
await window.addProjectTodo(projects[0].id, 'first todo')
await window.toggleProjectTodo(projects[0].id, projects[0].todos[0].id)

await window.insertPlanItem('2026-04-25', 'morning', 'try the lib', 'P2', 30)
await window.listPlanItems('2026-04-25')

await window.insertJournalEntry('a thought, right now')
await window.listJournalEntries({ limit: 5 })

await window.insertAdminReminder('renew passport')
await window.listAdminReminders()
const r = (await window.listAdminReminders())[0]
await window.markAdminReminderDone(r.id)

Available helpers

  • insertGoal(title) / listGoals()
  • insertProject(title, goalId?) / listProjects()
  • addProjectTodo(projectId, text) / toggleProjectTodo(projectId, todoId)
  • insertPlanItem(date, band, text, tier?, durationMin?) / listPlanItems(date)
  • insertJournalEntry(text, date?) / listJournalEntries({ limit? })
  • insertAdminReminder(text, remindOn?) / listAdminReminders() / markAdminReminderDone(id)

Heads-up: migrations 0003/0004/0005 must be applied (supabase db push) and the anon key must be filled into the inline config below, or every call will throw. Errors bubble — check the console.
